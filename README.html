* Game! Game! is a website that provides information about video games. Users can search video games and browse video games by categories. The website recommends games based on ratings and also provides similar games of the game users search. ## How To Run Run the python file flask_app.py to start the application. ## Features * The website displays a number of recommended games based on the user and critic ratings.
* The website displays the top three publishers, platforms and best-sellers based on the user and critic ratings.
* Users can browse video games by genre, platform, publisher, platform and sale record.
* Users can search for video games by name or publisher.
* The website provides a list of products by the same publisher as the searched video game.
* Users can connect with the developers through email. ## Citations of Dataset Video Game Sales - vgsales.csv (owned by Gregory Smith) Dataset page link: https://www.kaggle.com/gregorut/videogamesales Dataset Download Link: https://www.kaggle.com/account/login?ReturnUrl=%2fgregorut%2fvideogamesales%2fdownloads%2fvideogamesales.zip ## Features that are not implemented * Users can create personal accounts and manage their lists of favourite games. * The website can provide personalized recommendations of video games based on the usersâ€™ lists of favourite games. ## Authors * Yitong Chen * Megan Zhao * Eva Zhong
